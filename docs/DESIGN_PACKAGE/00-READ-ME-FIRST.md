Q: С чего начать кодить?
A: Читай 00_START_HERE.md → выбирай трек → открывай 03_MATHEMATICAL_SPECIFICATION.md

Q: Что такое "Hydraulic Diagnostic Platform"?
A: Симулятор гидравлики экскаватора CAT 320D + ML-модель поиска аномалий
   Цель: предсказывать отказы оборудования по данным датчиков

Q: Почему именно CAT 320D?
A: Промышленный стандарт, хорошая документация, 320D + D7/D8 покрывают 85% рынка

Q: Какие стандарты важны?
A: ISO 13849-1 (безопасность), ISO 6162 (гидравлика), ГОСТ 52138 (PLC)

Q: Где найти формулы для расчётов?
A: QUICK_REFERENCE.md → раздел "Formulas" + копируемые Python примеры

Q: Как организована кодовая база?
A: Смотри 04_SOFTWARE_ARCHITECTURE.md → раздел "Project Structure"
